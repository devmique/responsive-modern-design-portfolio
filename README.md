
# 🎨 Modern Responsive Portfolio Website (UI/UX Designer)

A **modern, elegant, mobile-first portfolio website** built with **React 18 + Vite + TypeScript**.  
Designed for UI/UX designers to showcase their work with style and professionalism.  

This project embraces clean typography, subtle animations, light/dark mode, and a mobile-first responsive layout.  

---

## ✨ Features

- 🌗 **Light / Dark Mode** — Seamless theme switching using `next-themes`
- 📱 **Mobile-First Responsive Design**
  - Sticky navbar with tabs: `About`, `Projects`, `Certifications`, `Contact`
  - Hamburger menu on smaller screens with vertical menu layout
- 📌 **Sticky Navbar** — Navigation always accessible
- 🧑 **About Section**
  - Profile image with subtle floating animation + active icon
  - Short bio with **typing animation** for UX Developer role
- 🧰 **Projects Section**
  - Scroll-in animated project cards
- 🏅 **Certifications Section**
  - Certification badges/logos in a responsive grid with fade-in animation
- 📬 **Contact Section**
  - Email + Message form
  - Sends messages directly to your personal email
- 🎯 **Smooth Animations**
  - Sections animate subtly when scrolled into view
- 💎 **Modern Design**
  - Minimal, elegant, with glassmorphism & soft shadows
  - Clean sans-serif typography
  - Smooth transitions and motion design

---

## 🛠️ Tech Stack

- ⚛️ **React 18 + Vite + TypeScript** — fast and modern frontend setup
- 🎨 **Tailwind CSS** — utility-first styling
- 🪞 **shadcn/ui (Radix)** — accessible UI components
- 🌗 **next-themes** — light/dark mode toggle
- 📡 **TanStack Query** — async state management
- 🔗 **React Router** — routing between sections
- 🔔 **Sonner** — toast notifications
- 🎯 **Lucide Icons** — modern icons
- 📊 **Recharts** — data visualization (if needed)

## 🚀 Getting Started

### 📦 Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/devmique/responsive-modern-design-portfolio.git
cd portfolio-website
npm install
````

### ▶️ Development Server

Start the dev server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### 🔨 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
